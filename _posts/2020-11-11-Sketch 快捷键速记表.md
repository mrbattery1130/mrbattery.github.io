## 自定义快捷键

系统偏好设置 > 键盘 > 快捷键

“应用程序”选择 Sketch ，“菜单标题”输入对应的名称，注意大小写。

![自定义 sketch 快捷键](https://tva1.sinaimg.cn/large/0081Kckwly1gkkmvqvyapj30ik0bmwfj.jpg)

下面是一些自定义快捷键参考，有些用户喜欢把上移一层（Cmd+]）和下移一层（Cmd+[）设置和 Photoshop 一样，大家根据自己喜好来定义就好。

| 自定义       | 快捷键  |
| :----------- | :------ |
| Vertically   | Cmd + \ |
| Horizontally | Cmd + ] |
| Top          | Cmd + ↑ |
| Right        | Cmd + → |
| Bottom       | Cmd + ↓ |
| Left         | Cmd + ← |
